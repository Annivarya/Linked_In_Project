Before running the code on the local machine

You must install the necessary liberaries:
1. Pandas
2. bs4 or BeautifulSoup
3. Selenium

Also download the geckodriver from the given link and set the path of your local machine as mentioned:

path:- "D:\\geckodriver\\geckodriver.exe"

link:- https://github.com/mozilla/geckodriver/releases

The code once run successfully will give you an excel file with the necessary details after scraping.
